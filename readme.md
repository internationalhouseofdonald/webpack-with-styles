## HMR (Hot Module Replacement)

When you think `loaders`, think `HMR`

> What other frameworks call HMR

* React: React Hot Loader
* Vue: Vue Loader
* Elm: Elm Hot Loader
* Redux: Redux HMR
* Angular: Angular HMR
